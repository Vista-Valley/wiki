---
title: Quirks
description: Unique modifications to Vanilla gameplay that will enhance your experience in Vista Valley.
---

## General

* Leaf decay much faster on trees after all the wood has been broken.
* Stonecutters can deal damage to entitites on top of it.
* Players can build elevators to traverse between floors!
  * Drop an enderpearl onto a wool block of any color to create an elevator block. Then do the same process to another wool block at the same X and Z coordinates, but different Y coordinates. You can now jump or sneak to go up or down respectively.
  * Player is required to be [**2** 🌟](/earth/gameplay/stars/#star-2) to use this feature.

## Mobs

* Some animals might be aggressive to players.
* Feeding mushrooms to cows can turn them into mooshroom cows.
* Change a fox's type by feeding it a white tulip (snow fox) or an orange tulip (regular fox).
* Polar bears can be bred by feeding them salmon.
* Right Click a shulker with a dye to change its color.
* Phantoms grow larger and stronger the longer a player goes without sleeping.
* Phantoms ignore players holding a torch.
* Villagers and Wandering Traders can be leashed.
* Iron Golems can be calmed by healing them or gifting them a poppy.
* Axolotls can breed when fed tropical fish items, not just tropical fish buckets.
* Axolotls no longer attack passive mobs. (They will still go for hostile mobs!)
* Players can silence annoying mobs by naming them `silence me` or `silence_me` using name tag.

## Mob Drops (Temporarily Disabled)

* Turtles have a 35% chance to drop 1-2 sea pickles.
* Turtles have a 10% base chance to drop 1 to 2 turtle scutes, scaling up to 16% with Looting III.
* Silverfish drop 0 to 2 cracked stone bricks, increasing by 1 per Looting level.
* Silverfish drop 0 to 1 cobblestone.
* Silverfish have a 15% chance to drop 1 to 2 iron nuggets.

## Loot Tables

* Budding amethyst blocks can be picked up using tools with silk touch.
* Reinforced deepslate blocks can be picked up using tools with silk touch.
* Spawner blocks can be picked up using tools with silk touch.
  * Player is required to be [**1** 🌟](/earth/gameplay/stars/#star-1) to use this feature.
* Four secret paintings can be obtained if you get lucky.

## Tools

* Pickaxes can break light source blocks (glowstone, sea lanterns, etc.) faster.
* Pickaxes can break glass blocks and panes faster.
* Pickaxes can break break reinforced deepslate blocks faster.
* Hoes can break froglights faster.
* Hoes can break castus faster.
* Axes can break skulls and heads faster.

## Enchantment Table

* Players can store lapis lazuli in the enchantment table indefinitely.
* Chiseled bookshelves can contribute to the total enchant power like normal bookshelves.
* Most transparent/non-full blocks (fences, carpets, banners, etc.) can be used in the enchant area without taking enchant power away.

## Nether

* Player-made explosions have been disabled.
* Players may use crying obsidian when building a nether portal.
* Players can create non-rectangular nether portals.
* Building blocks above the height limit of the world is not allowed.

## The End

* Player-made explosions have been disabled.
* Players may change color of shulker by right-clicking it with a dye item.
