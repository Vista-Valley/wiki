---
title: Warps 
---

Warps allow players to quickly travel to different locations on the server.
These locations can be public (like PvP arenas, shops, XP farms) or private (personal bases with access control).

:::caution[Player Inactivity]
Players with more than 60 days of inactivity will have their warps automatically deleted. Please make sure to log in at least once every 60 days to keep your warps active.
:::

---

##  Commands

| Command | Description |
| ------- | ----------- |
| `/warp <warp-name>` | Teleport to a warp |
| `/warp list` | View all available warps |
| `/warp info` | Get details about a warp |
| `/warp help` | Show all warp commands |
| `/warp create <warp-name>` | Creates a new warp at your current location; costs `$5,000`. |
| `/warp remove <warp-name>` | Deletes the warp permanently. |
| `/warp rename <old-name> <new-name>` | To change the owner of the warp. |
| `/warp reset <warp-name>` |  Resets the warp location.|
| `/warp set <warp-name> <password>` | Protects your warp with a password. |
| `/warp cost set <warp-name> <cost>` | Set a fee for using your warp; maximum fee is `$1,000,000`. |
| `/warp cost remove <warp-name>` | Remove the fee for using your warp. |
| `/warp lock <warp-name>` | Locks the warp |
| `/warp open <warp-name>` | Unlocks the warp |
| `/warp ban <warp-name> <player name>` | Prevents a player from accessing your warp. |
| `/warp password  set <password>` | Prevents a player from accessing your warp. |
| `/warp rate <warp-name> <rating>` | Rate a warp (up to 5). |
| `/warp favourite <warp-name>` | Adds the warp to your favorites list. |
| `/warp setowner <warp-name> <player>` | Transfers ownership of the warp. |

---

##  Common Uses of Warps

Warps are commonly used for:
- ⚔️ PvP Arenas
- 🛒 Shops & Trading Halls
- ⚡ XP Farms
- 🏠 Personal Bases
- 🎮 Custom Game Areas
