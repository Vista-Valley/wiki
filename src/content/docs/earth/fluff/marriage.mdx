The **marriage** system lets players marry each other in Minecraft! A family tree system will be added in a later update.

### Commands

| Command | Description |
| --- | --- |
| `/family` | View your relationship status |
| `/family <playername>` | View someone's relationship status |
| `/family propose <playername>` | Propose to a player |
| `/family accept <playername>` | Accept a player's proposal |
| `/family deny <playername>` | Decline a player's proposal |
| `/family ring <playername>` | Buy a wedding ring for a player |
| `/family divorce` | Divorce your partner |
