:::note[Star Requirement]
Player must be at least [**20** 🌟](/earth/gameplay/stars/#star-20) to access this feature.
:::

**ImageFrame** is a feature that allows players to display custom images in-game using item frames and maps. Players can create an ImageFrame by providing a name, image URL, width, and height. The image will be split into multiple maps if necessary, and players can then place these maps in item frames to display the full image.

## How to Use ImageFrame

The two main ways to create an ImageFrame are:

1. Type `/imageframe create <name> <url> <width> <height>` <br />
   to get multiple parts of the image if either width or height is larger than 1.
2. Type `/imageframe create <name> <url> <width> <height> combined` <br />
   to get a single combined map no matter the size.

:::note[Hints]
- `<name>` is your preferred name for the image. For example, `cute_cat`.
- `<url>` is the image link of your choice.
- `<width>` is the width of the final ImageFrame product. Maximum is `5`.
- `<height>` is the height of the final ImageFrame product. Maximum is `5`.
:::

To delete your own images, type `/imageframe delete <name>`.

To view all your images, type `/imageframe list`.

:::caution[Rank Limits]
Currently, at [**`20`** 🌟](/earth/gameplay/stars/#star-20), the limit is *one* ImageFrame. Sponsors get to upload more images on the server: Topaz has a limit of 5, Citrine has a limit of 10, and so on.
:::
