## Money

Money is the main currency in this server. It can be used to promote your star ranks, purchasing items from the item `/shop`, trading with other players, and much more! Send money to other players using the following command: `/pay [playerName] [amount]`.

### Earning Money <a href="#making-money" id="making-money"></a>

In order to contribute to upkeep, or create a town, players must earn money. There are several ways a player can do this:

* Voting for the server,
* Joining piñata parties,
* Selling items at the `/shop`,
* Trading with other players.

You can check your current balance with `/balance`. Upon joining the server for the first time, players start with `$5 000`, and a few basic items to assist with the early game experience.

### Voting the Server

[Vote](/home/vote/) for the server to get Basic Capsule keys in return! Unlocking these capsules may award you with money and other rare rewards. This is by far the simplest way to earn money and/or experience fast!

### **Joining the Piñata** Party

The piñata is summoned after `50` votes have been casted in total. Hitting the piñata when it's around will randomly award the player with [**Piñata Pouches**](#user-content-fn-1)[^1]. After the piñata has been hunted, all online players will receive a **Piñata Capsule** key as a bonus reward!

### Selling Items at the Shop

Sell your items at the item `/shop` to earn money! The item shop is the main source of earning lots of money in the server. To access the item shop, simply type the following command in-game: `/shop`.

***

## Glims

**Glims** are the freemium currency in this server that can be used to purchase rare **capsule keys**, **spawners**, **cosmetics**, and more! Send glims to other players using the following command: `/glims pay [playerName] [amount]`.

### Earning Glims <a href="#making-money" id="making-money"></a>

Glims are scarce and especially valuable in the server as a recently introduced currency! There are several ways a player can earn glims:

* Voting the server
* Hang out at the **Glim Hub**
* Trading with other players

You can check your current balance with `/glims`.

***

## Gems

**Gems** are the premium currency in this server that can be used to purchase rare **capsule keys**, **spawners**, **cosmetics**, and more! Send gems to other players using the following command: `/gems pay [playerName] [amount]`.

### Obtaining Gems

**Gems** are mainly obtained via purchases from the [Vista Store](https://store.vistavalley.xyz/). You can check your current balance with `/gems`.

[^1]: These pouches contain money, experience points, glims, capsule keys, Heart of the Sea, Heavy Core, Elytra, Netherite Scrap and Netherite Ingot.
