---
title: Auction House
---

The **Auction House** is a player-run marketplace where you can buy and sell items using multiple currencies, including **money**, **experience points**, **Glims**, and **Gems**. It functions similarly to the server shop but with the added benefit of a variety of unique items from other players. You can browse listings and purchase items directly, making it a fantastic way to find rare goods and contribute to the player-driven economy.

## Commands

| Command | Description |
| ------- | ----------- |
| `/auction expired` | View a menu showing your expired listings. |
| `/auction history` | Check your sales history in a menu. |
| `/auction open` | Open the main Auction House menu. |
| `/auction sell <price>` | Put the item you're holding up for auction. |
| `/auction selling` | View a menu displaying your active listings. |
| `/auction unclaimed` | Open a menu to see your unclaimed listings. |
