---
title: Player Trading
---

Upon reaching [**7** 🌟](/earth/gameplay/stars/#star-7), players can engage in direct trading with one another. 
This allows for a more personalized and flexible trading experience, 
enabling players to negotiate and exchange items without the need for an intermediary platform like the Auction House.
Furthermore, players can also <abbr title="Default for crouch">Shift</abbr> + Right Click another player to send or accept a trade request.

## Commands

| Command | Description |
| ------- | ----------- |
| `/trade <player>` | Send a trade request to a player. |
| `/pay <player>` | Alias for `/trade <player>`. |
| `/trade toggle` | Toggle receiving trade requests from players. |
