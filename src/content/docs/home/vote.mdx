Vote for the server on these platforms to support the server and earn rewards in return. You can earn voting rewards for up to **10 votes per day** across all platforms.

:::tip
**Votes 1 to 5 (Daily):** Each vote guarantees a **Basic Capsule** key and **5 Glims**.

**Votes 6 to 10 (Daily):** Each vote guarantees **5 Glims** and a chance to win one additional key. The system rolls for these keys in sequence until you win one or all rolls fail:
* A 50% chance for a **Basic Capsule** key.
* A 25% chance for a **Piñata Capsule** key.
* A 12.5% chance for a **Mystic Capsule** key.

These extra voting sites help boost our server rankings and serve as reliable backups if any primary platforms go offline.
:::

:::note[Monthly Top Voter Competition]
At the end of each month, the top ten voters on [Minecraft-MP](https://minecraft-mp.com/server-s333194) will receive **500 glims** as a bonus reward. On top of that, the top voter of the month will be awarded an exclusive flair and a **Piñata Spawn Egg** that starts a **Piñata Party** upon activation!

If the top voters have the same amount of votes, they will all be equally awarded.
:::

## Piñata Party

The piñata is summoned after `50` votes have been cast in total. Hitting the piñata when it's around will randomly award the player with **Piñata Pouches**. After the piñata has been hunted, all online players will receive a **Piñata Capsule** key as a bonus reward!

:::note
These pouches contain money, experience points, glims, capsule keys, Heart of the Sea, Heavy Core, Elytra, Netherite Scrap, and Netherite Ingot.
:::
